#### 第702关 第2关：隐性之钥

**教学目标**：学习三链数（Naked Triple）：同一区域内三个格子恰好只包含3个不同数字的组合，可以排除该区域其他格子中的这3个数字。

**开场对话**

**触发器提示文案**

# 第三部分：技巧教程文案

以下是各高级解题技巧的手把手教程内容，以守笼人和阿岩的对话形式呈现。

## 🔐 显性数对 Naked Pair

**步骤1（守笼人）**

📖 今天学第一个秘术：显性数对。格子里的小数字叫「候选数」——就是这个格子可能填什么。比如某个格子里写着3和7，意思是：「这里要么填3，要么填7，不可能是别的」。💡 小数字 = 候选数 = 可能的答案

**步骤2（守笼人）**

第一步：找到第一个特殊格子。👉 看这里！D5（绿色大圈圈住的格子）。它的候选数只有两个：3 和 7！没有别的可能了，D5要么是3，要么是7。

**步骤3（守笼人）**

第二步：找到它的搭档！👉 再看旁边的D6（第二个绿圈）。它的候选数也是只有两个：3 和 7！D5={3,7}，D6={3,7}——两个格子候选数完全一样！

**步骤4（守笼人）**

第三步：这两个格子把3和7「锁」住了！D行里，3要出现一次，7也要出现一次。但能填3和7的地方，只有D5和D6这两个格子！🔒 所以：D5和D6一个是3，另一个就是7——跑不掉了！💡 就像两个座位，两个人已经坐定了，其他人不能再来坐了

**步骤5（守笼人）**

第四步：排除D2里的3！👉 看D2（红箭头指的格子）。D2的候选数是：~~3~~,4,7,9。但3已经被D5/D6锁住了，D行其他地方不可能再有3！划掉D2里的3！

**步骤6（守笼人）**

第五步：再划掉D2里的7！7也被锁住了！D2的候选数现在是：~~3~~,4,~~7~~,9。7也划掉！D2只剩下4和9了。

**步骤7（阿岩）**

第六步：别忘了同宫也能排除！D5和D6不仅在同一行，还在同一个宫（中央宫，紫色框）里！👉 看F4——它也在这个宫里。F4候选数有~~3~~,5,7。3被锁住了，划掉F4里的3！

**步骤8（守笼人）**

✨ 哇！直接出数了！排除后候选数变少，继续推理就能出数——关键是你学会了：数对可以排除其他格子的数字！

**步骤9（守笼人）**

🔐 显性数对口诀：🔍 找——同一行/列/宫里，两个格子候选数完全相同（就是{a,b}）；🔒 锁——这两个格子一定是a和b，跑不掉；❌ 删——该行/列/宫其他格子里的a和b全部划掉；✨ 填——划到某个格子只剩一个数→直接填入！一个数对往往能解开一片区域！现在去题里找数对破解它吧！

## 👁️ 隐性数对 Hidden Pair

**步骤1（守笼人）**

👁️ 第二个秘术：隐性数对。显性数对是：「两个格子只能填a和b」。隐性数对要反过来看：「数字a和数字b，只能填在某两个格子里」。💡 一个看格子，一个看数字——方向相反！

**步骤2（守笼人）**

第一步：聚焦中央宫。👉 看紫色框的中央宫（第5宫）。这个宫已经填了5个数字：8、6、9、2、4。还缺4个数字：1、3、5、7，有4个空格。

**步骤3（守笼人）**

第二步：数字3能去哪？反过来看：数字3在这个宫里能放在哪些格子？E5？E行有E1=3（红圈），同E行不能有两个3→E5≠3。F4？C4列有A4=3（红圈），同列不能有两个3→F4≠3。所以3只能去 D5 或 F6！

**步骤4（阿岩）**

第三步：数字7也只能去那！同样看数字7：E5？E行有E9=7（红圈）→E5≠7。F4？C4列有I4=7（红圈）→F4≠7。7也只能去 D5 和 F6！

**步骤5（守笼人）**

第四步：隐性数对找到了！3和7都只能去D5和F6这两个格子！虽然D5候选看起来是{1,3,5,7}、F6是{1,3,5,7}——但1和5不可能在这两个格子！因为3和7已经把位置「占」了！

**步骤6（守笼人）**

第五步：清理多余候选数。把D5和F6里除了3和7之外的数字全部删掉！D5的 ~~1~~ 和 ~~5~~ 划掉；F6的 ~~1~~ 和 ~~5~~ 也划掉。隐性数对被「挖」出来后，就变成了显性数对 {3,7}！

**步骤7（阿岩）**

✨ 剩下的格子直接出数！现在D5和F6是{3,7}数对，中央宫剩下E5和F4要填1和5。E5={1,5}，F4={1,5}——又形成一个数对！而且直接就能推出E5=1、F4=5！

**步骤8（守笼人）**

👁️ 隐性数对口诀：👁️ 反着看——不看「格子能填什么」，看「数字能去哪里」；🔍 找挤压——a和b都只能去同样两个格子→隐性数对；🧹 清理——这两个格子里其他候选数全部删除。隐性数对虽然难找，但一旦找到，清理效果很强！

## 🔱 三链数 Naked Triple

**步骤1（守笼人）**

🔱 第三个秘术：三链数。理解了数对（2个格子锁2个数），三链数就是自然延伸——三个格子锁住三个数字！

**步骤2（守笼人）**

第一步：找到三个格子。👉 看D行三个橙色格子：D4候选：{2,4}；D5候选：{4,9}；D6候选：{2,9}。三个格子候选数加起来，一共只有 2、4、9 三个数字！💡 不要求每个格子都有三个数！加起来是三个就行

**步骤3（阿岩）**

第二步：三个数字被锁住了！2、4、9这三个数字被这三个格子包圆了！不管怎么分配（有6种排列可能），D4,D5,D6一定是2,4,9的某种排列。🔒 其他格子不能再有2、4、9了！

**步骤4（守笼人）**

第三步：排除同行其他格子！D行其他格子里的2、4、9全部划掉：👉 D2的 ~~4~~ 划掉；👉 D8的 ~~4~~ 划掉；👉 D9的 ~~2~~ ~~9~~ 划掉

**步骤5（守笼人）**

🔱 三链数要点：🔱 条件——三个格子候选数总共只涉及3个数字；🔒 锁住——这3个数字一定在这3个格子里；❌ 删除——其他格子中的这3个候选数全排除。⚠️ 每个格子不需要都有全部3个候选！同理还有「四链数」（四格锁四数）。

## ✈️ X-Wing X翼构型

**步骤1（守笼人）**

✈️ 第四个秘术：X-Wing（X翼构型）。名字来自《星球大战》的X翼战斗机——形状像个X。X-Wing看的不是「格子能填什么」，而是某个数字能去哪里。以数字 7 为例。

**步骤2（守笼人）**

第一步：找两行中数字7的位置。👉 看D行和F行（两条橙色行带）。D行其他格子都填满了，数字7只能放在 D2 或 D5！F行也一样——7只能放在 F2 或 F5！

**步骤3（守笼人）**

第二步：四个格子形成矩形！D2、D5、F2、F5 四个格子正好形成矩形！只有两种可能：✈️ D2=7 → F5=7（左上到右下对角线）；✈️ D5=7 → F2=7（右上到左下对角线）。红色X就是这两种可能——7一定在其中一条对角线上！

**步骤4（阿岩）**

第三步：排除两列其他位置的7！不管走哪条对角线，C2列和C5列都一定会有7！所以这两列其他格子都不能是7：👉 I2的 ~~7~~ → 划掉；👉 G5的 ~~7~~ → 划掉

**步骤5（守笼人）**

✈️ X-Wing口诀：✈️ 找矩形——某数字在两行各只有2个位置，恰好在相同两列；❌ 删列——该两列其他格子的此数字全排除；🔄 双向——两列×两行也成立→删两行。X-Wing是「鱼形」技巧基础，剑鱼(Swordfish)就是3×3升级版！

---

## 🐟 Swordfish 剑鱼构型

**步骤1（守笼人）**

🐟 第五个秘术：Swordfish（剑鱼）。理解了X-Wing（2×2矩形），剑鱼就是升级版——3行×3列的「渔网」。原理一模一样，只是从2行2列变成了3行3列！

**步骤2（守笼人）**

第一步：找三行中数字5的位置。以数字5为例。👉 看C、F、H三行（橙色行带）：C行：5只能去 C2 或 C5；F行：5只能去 F2 或 F7；H行：5只能去 H5 或 H7。全都落在第2、5、7列三列中！

**步骤3（守笼人）**

第二步：3×3的渔网形成了！这6个格子形成了一个3×3的渔网！虽然组合比X-Wing多（有多种分配方式），但不管5怎么分配，这三列每列都一定会有一个5。🐟 5被这张三列的网「兜住」了！

**步骤4（阿岩）**

第三步：排除三列其他位置的5！这三列的其他所有格子都不能是5！👉 红叉标记的格子，~~5~~全部划掉：E2的5→划掉；A5、I5的5→划掉；B7的5→划掉

**步骤5（守笼人）**

🐟 剑鱼要点：🐟 找网——某数字在三行各有2~3个位置，全部落在相同三列；❌ 删列——该三列其他格子的此数字全排除。原理和X-Wing一模一样，只是2×2变成了3×3！还有4×4的「水母」(Jellyfish)，原理相同但更少见。

## 二、guide.js —— 教学模式交互文案

### 2.1 错误提示 / 降级

| 场景 | 文案 |
|---|---|
| 教学关卡加载失败（alert） | `关卡加载失败，请返回重试` |
| 降级标题（无 name/title） | `教学关卡 ${N}` |
| 教学目标前缀 | `🎯 ${teachingGoal}` |
| 通关计时 | `用时 ${m}分${s}秒` |
| 星级 | ⭐⭐⭐（根据用时 1~3 星） |

| 提示按钮给答案 Toast | `🎯 答案是 ${num}，填入后会自动排除相关候选数` |

### 2.2 45 法则计算器（10/21/45 自适应）

- 动态标题：`🧮 ${ruleName}计算器`（ruleName：`10法则` / `21法则` / `45法则`，取决于盘面大小）
- 错误/空结果文案：
  - `请输入有效的格子数（1-${size}）和目标和（1-${ruleSum}）`（输入超出范围）
  - `必含数字数量不能超过格子数`
  - `必含数字不能同时被排除`
  - `没有符合条件的组合`
- 结果展示：
  - 每条组合后缀：`= ${sum}`
  - 组合计数：`${N} 种`（含初始/无效状态 `0 种`）

### 2.3 连填模式（长按数字键激活）

- 激活 Toast：`⚡ 连填${num}：直接点空格快速填入`
- 填满自动关闭 Toast：`✅ ${num} 已填满，连填自动关闭`

### 2.4 重来确认

- confirm 对话框：`确定要重来这关吗？所有已填的数字将被清空。`

### 2.5 猜错提示

### 6.4 X-Wing X 翼构型

- **title**：`X-Wing X翼构型`
- **badge**：`✈️`

**Step 1 —— 守笼人**
- **hint**：`X-Wing看某个数字能去哪里（以7为例）`
- **text**：

```html
<span class="look">✈️ 第四个秘术：X-Wing（X翼构型）</span><br><br>名字来自《星球大战》的X翼战斗机——形状像个X。<br><br>X-Wing看的不是"格子能填什么"，而是<b>某个数字能去哪里</b>。<br><br>以数字 <span class="big">7</span> 为例。
```

**Step 2 —— 守笼人**
- **hint**：`D行和F行中，7各只有两个位置！`
- **text**：

```html
<span class="look">第一步：找两行中数字7的位置</span><br><br><span class="here">👉 看D行和F行</span>（两条橙色行带）。<br><br>D行其他格子都填满了，数字7只能放在 <b>D2</b> 或 <b>D5</b>！<br>F行也一样——7只能放在 <b>F2</b> 或 <b>F5</b>！
```

**Step 3 —— 守笼人**
- **hint**：`X形对角线——7一定在其中一条上！`
- **text**：

```html
<span class="look">第二步：四个格子形成矩形！</span><br><br>D2、D5、F2、F5 四个格子<b>正好形成矩形</b>！<br><br>只有两种可能：<br>✈️ <b>D2=7 → F5=7</b>（左上到右下对角线）<br>✈️ <b>D5=7 → F2=7</b>（右上到左下对角线）<br><br>红色X就是这两种可能——7一定在<b>其中一条对角线</b>上！
```

**Step 4 —— 阿岩**
- **hint**：`两列其他格子都不能是7了！`
- **text**：

```html
<span class="look">第三步：排除两列其他位置的7！</span><br><br>不管走哪条对角线，<b>C2列和C5列都一定会有7</b>！<br><br>所以这两列<b>其他格子</b>都不能是7：<br><br>• <span class="here">👉 I2</span>的 <span class="strike">7</span> → 划掉<br>• <span class="here">👉 G5</span>的 <span class="strike">7</span> → 划掉
```

**Step 5 —— 守笼人（口诀总结）**
- **hint**：`X-Wing：矩形→对角线→排除列（或行）`
- **text**：

```html
<span class="look">✈️ X-Wing口诀</span><br><br>✈️ <b>找矩形</b>：某数字在<b>两行</b>各只有2个位置，恰好在<b>相同两列</b><br>❌ <b>删列</b>：该两列其他格子的此数字全排除<br>🔄 <b>双向</b>：两列×两行也成立→删两行<br><br>X-Wing是"鱼形"技巧基础，剑鱼(Swordfish)就是3×3升级版！
```

### 6.5 Swordfish 剑鱼构型

- **title**：`Swordfish 剑鱼构型`
- **badge**：`🐟`

**Step 1 —— 守笼人**
- **hint**：`剑鱼是X-Wing的3×3升级版`
- **text**：

```html
<span class="look">🐟 第五个秘术：Swordfish（剑鱼）</span><br><br>理解了X-Wing（2×2矩形），剑鱼就是升级版——<b>3行×3列的"渔网"</b>。<br><br>原理一模一样，只是从2行2列变成了3行3列！
```

**Step 2 —— 守笼人**
- **hint**：`三行中的5都只出现在同样三列里！`
- **text**：

```html
<span class="look">第一步：找三行中数字5的位置</span><br><br>以数字<b class="big">5</b>为例。<span class="here">👉 看C、F、H三行</span>（橙色行带）：<br>• C行：5只能去 <b>C2</b> 或 <b>C5</b><br>• F行：5只能去 <b>F2</b> 或 <b>F7</b><br>• H行：5只能去 <b>H5</b> 或 <b>H7</b><br><br>全都落在<b>第2、5、7列</b>三列中！
```

**Step 3 —— 守笼人**
- **hint**：`3×3的网罩住了三列！`
- **text**：

```html
<span class="look">第二步：3×3的渔网形成了！</span><br><br>这6个格子形成了一个<b>3×3的渔网</b>！<br><br>虽然组合比X-Wing多（有多种分配方式），但不管5怎么分配，这三列<b>每列都一定会有一个5</b>。<br><br>🐟 5被这张三列的网"兜住"了！
```

**Step 4 —— 阿岩**
- **hint**：`三列其他格子的5全部排除！`
- **text**：

```html
<span class="look">第三步：排除三列其他位置的5！</span><br><br>这三列的<b>其他所有格子</b>都不能是5！<br><br><span class="here">👉 红叉标记的格子</span>，<span class="strike">5</span>全部划掉：<br>• E2的5 → 划掉<br>• A5、I5的5 → 划掉<br>• B7的5 → 划掉
```

**Step 5 —— 守笼人（要点总结）**
- **hint**：`剑鱼：三行三列的网！`
- **text**：

```html
<span class="look">🐟 剑鱼要点</span><br><br>🐟 <b>找网</b>：某数字在<b>三行</b>各有2~3个位置，全部落在<b>相同三列</b><br>❌ <b>删列</b>：该三列其他格子的此数字全排除<br><br>原理和X-Wing一模一样，只是2×2变成了3×3！<br><br>还有4×4的"水母"(Jellyfish)，原理相同但更少见。
```
